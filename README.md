# DevOps Visit Counter App 🚀

אפליקציית Full-Stack מבוססת **Flask** ו-**Redis** המדגימה ארכיטקטורת מיקרו-שירותים (Microservices) ופריסה מודרנית בענן.

## 🛠 טכנולוגיות בשימוש
* **Backend:** Python Flask
* **Database:** Redis (לניהול מונה ביקורים בזמן אמת)
* **Infrastructure:** Docker & Docker Compose
* **Deployment:** Render Cloud

## 🚀 איך להריץ מקומית?
כדי להריץ את כל המערכת במחשב שלכם, וודאו ש-Docker מותקן והריצו מהתיקייה הראשית:
```bash
docker-compose up --build
